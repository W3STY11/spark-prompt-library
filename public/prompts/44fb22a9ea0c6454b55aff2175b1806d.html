⚙️ What This Mega-Prompt Does:

● Provides a framework for designing surveys that capture genuine sentiment in influencer marketing campaigns.
● Guides in creating platform-specific questions that resonate with influencer audiences.

● Incorporate interactive components like polls, sliders, and emoji reactions to keep the survey experience dynamic and engaging.

● Balance quantitative metrics with qualitative insights to gain a comprehensive understanding of audience sentiment and behavior.

#RESPONSE GUIDELINES:
1. Start with a Survey Purpose Statement that frames the research goal in influe

2. Design Pre-Survey Audience Segmentation questions to filter respondents appro

3. Create Core Engagement Questions that measure authentic connection without fe

Each section should:
- Use native platform language and current slang appropriately
- Avoid marketing jargon that triggers skepticism
- Include visual elements or interactive components where possible
- Balance quantitative metrics with qualitative insights
- Respect audience time with progressive disclosure
- Build trust before asking sensitive questions

#SURVEY DESIGN CRITERIA:
1. Questions must feel conversational, not interrogational
2. Use platform-native formatting (polls, sliders, emoji reactions)
3. Maximum 3-5 minutes completion time for core survey
4. Include "skip logic" to personalize question flow
5. Avoid leading questions that bias toward positive brand sentiment
6. Test for both stated preferences and revealed behaviors
7. Include attention-check questions disguised as content
8. Offer value exchange (insights, rewards, exclusive content)
9. Design for mobile-first experience
10. Build in social proof elements without compromising anonymity

Focus on:
- Authentic engagement metrics over vanity metrics
- Community sentiment over individual opinions
- Behavioral patterns over stated intentions
- Platform-specific communication styles

Avoid:
- Corporate survey language
- Questions about "brand awareness" or "purchase intent"
- Likert scales without context
- Demographic questions that feel invasive
- Assumptions about influencer audience sophistication
