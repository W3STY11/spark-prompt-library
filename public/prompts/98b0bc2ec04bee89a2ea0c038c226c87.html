#RESPONSE GUIDELINES:
1. Begin with an introduction stating the specific unit or topic and the academic subject for which formative assessment questions are being developed.
2. Specify the types of questions to include in the assessment (e.g., multiple choice, short answer, true/false, fill-in-the-blank, essay).
3. Indicate the difficulty levels to assign to each question (e.g., easy, medium, hard).
4. State the total number of formative assessment questions to generate.
5. List the Bloom's Taxonomy levels to target with the questions (e.g., remembering, understanding, applying, analyzing, evaluating, creating).
6. Present the questions and their answer keys in a table format with the following columns:
   - Question
   - Type
   - Difficulty
   - Answer
   - Bloom's Level
   - Source
7. Cite any sources used in creating the questions.

#TASK CRITERIA:
1. The prompt should guide the AI to generate questions specifically tailored to the given unit and subject.
2. The generated questions should cover a range of difficulty levels and target various Bloom's Taxonomy levels to assess different aspects of student understanding.
3. The prompt should instruct the AI to provide answer keys for each question and cite any sources used in creating the questions.
4. The prompt should be clear, concise, and actionable, avoiding unnecessary explanations or digressions.
