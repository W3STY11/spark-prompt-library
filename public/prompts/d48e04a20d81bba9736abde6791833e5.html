⚙️ What This Mega-Prompt Does:

● Develops a comprehensive methodology for tracking brand mentions across various social media platforms, detailing monitoring approaches, data collection methods, and relevant metrics.
● Designs an interactive, real-time dashboard that visualizes key metrics and engagement data, including layout sketches, key visualizations, and interactive features.
● Provides actionable insights and recommendations based on analyzed data to support data-driven decision-making.

💡Tips:

● Develop a standardized monitoring template that can be customized for each social media platform, ensuring consistency in data collection while allowing flexibility for platform-specific metrics and tools.

● Integrate real-time alert systems in the dashboard design to notify the user of significant spikes or drops in brand mentions, enabling immediate response to potential PR crises or viral content opportunities.

● Regularly review and update the set of metrics and visualizations based on emerging trends and changes in social media algorithms to maintain the relevance and effectiveness of the monitoring system and dashboard.
