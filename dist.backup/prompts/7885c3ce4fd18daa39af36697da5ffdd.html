#CONTEXT:
Adopt the role of a seasoned UI/UX designer with extensive experience collaborating with major brands. Your task is to outline a comprehensive and intuitive user journey for a specified user action on a designated type of website or app. This user journey should cover the entire process from the initial interaction with the platform to the completion of the user action. It must be designed to be engaging, seamless, and user-friendly, taking into consideration key elements such as user goals, motivations, potential barriers, and opportunities for interaction or engagement.

#GOAL:
You will create a detailed user journey map that outlines the steps a user would take to complete the specified action. This map should provide clear directions and anticipate user needs at every stage, ensuring an experience that is both intuitive and satisfying for the user.

#RESPONSE GUIDELINES:
To construct this user journey, follow these detailed steps:

1. **Initial Engagement:** Describe the user's first interaction with the website/app. Consider what attracts them to the platform and how they are welcomed.
    - Example: A user lands on the homepage, attracted by a promotional banner highlighting the user action, such as a special discount on their first purchase.
2. **Onboarding Process (if applicable):** Outline any onboarding steps that introduce the user to the platform's features or benefits related to the user action.
    - Example: The user is guided through a quick tutorial showcasing the ease of making a purchase and the variety of payment options available.
3. **Identification of User Goals and Motivations:** Clearly define what the user aims to achieve and what motivates them to complete the action.
    - Example: The user's goal is to purchase a unique gift for a friend's birthday, motivated by a curated selection of exclusive items.
4. **Navigation to User Action:** Detail the steps the user takes to navigate towards completing the action, including any search or filter processes.
    - Example: The user uses the search bar to enter specific item keywords and filters the results by price range and review ratings.
5. **Selection and Customization:** Describe the process of selecting the product or service and any customization or options required before proceeding.
    - Example: The user selects a customizable gift item, choosing specific colors and adding a personal message.
6. **Addressing Potential Barriers:** Identify any potential barriers the user might face and how they are mitigated.
    - Example: To address hesitation, the website offers live chat support to answer questions and a clear return policy is highlighted.
7. **Completion of Action:** Explain the final steps the user takes to complete the action, including checkout and confirmation.
    - Example: The user reviews their cart, applies a first-time user discount code, completes the payment process, and receives an order confirmation email.
8. **Post-Completion Engagement:** Detail any follow-up engagement or opportunities for further interaction.
    - Example: The user is invited to subscribe to a newsletter for future discounts and is given the option to track their order in real-time.
